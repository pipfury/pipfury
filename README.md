# PipFury Evidence Desk

Astro + Tailwind + MDX authority website for evidence-first forex education.

## Local use

```bash
npm install
npm run dev
npm run build
```

## Deployment

Deploy the repository to Cloudflare Pages.
Build command: `npm run build`
Output directory: `dist`

## Content model

- `src/content/learn` for lessons
- `src/content/analysis` for educational chart case files
- `src/content/journal` for transparent reviews
- `src/content/tools` for risk and execution tools
- `src/content/resources` and `src/content/glossary` for reference assets
- `src/content/legal` for disclosure pages

No signals, no financial advice, no performance guarantees.
