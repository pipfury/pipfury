## Core labels
EFFORT, NO RESULT, LIQUIDITY RAID, TEST, INVALIDATION, ABSORPTION, WEAK CLOSE, BACKGROUND STRENGTH, BACKGROUND WEAKNESS.

## Label rule
One label should teach one idea. Over-annotated charts become noise.
